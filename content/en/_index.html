{{< /blocks/cover >}} {{% blocks/lead color="secondary" %}} Codify Policies for Infrastructure as Code As you embrace Infrastructure as Code (IaC) such as [Terraform](https://www.terraform.io/), [Kubernetes](https://kubernetes.io/), [Argo CD](https://argo-cd.readthedocs.io/en/stable/), [Atlantis](https://www.runatlantis.io/) and AWS CloudFormation, it is important to ensure that security best practices and compliance requirements are observed. Terrascan provides 500+ out-of-the-box policies so that you can scan IaC against common policy standards such as the CIS Benchmark. It leverages the [Open Policy Agent](https://www.openpolicyagent.org/) (OPA) engine so that you can easily create custom policies using the Rego query language. {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fas fa-code" title="Get Started" url="https://runterrascan.io/docs/getting-started/" %}} Follow the documentation to get scanning as quickly as possible! {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-github" title="Contributions welcome!" url="https://github.com/accurics/terrascan" %}} We do a [Pull Request](https://github.com/accurics/terrascan/pulls) contributions on **GitHub**.
